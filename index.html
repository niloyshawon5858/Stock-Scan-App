About
StockScan — Free barcode scanner for inventory management.
Works offline. Data syncs to Google Sheets when connected.

Barcode scanning powered by ZXing.
No server. No subscription. Fully open-source.
Setup instructions
1. Open Google Sheets → Tools → Apps Script
2. Paste the Apps Script code (see below)
3. Deploy → New deployment → Web app
4. Set access to "Anyone"
5. Copy the URL into settings above
